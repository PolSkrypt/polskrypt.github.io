Rich Text Editor JavaScript Library

Lightweight javascript library without any dependencies that will help you to easily create your own modern Rich Text Editor with many advanced editing tools.

Sample

Below you can try the Rich Text Editor made with this JavaScript library:

Installation

To install this Rich Text Editor JavaScript Library on your web page or web app, paste the following line in <head> section (just before the closing </head> tag):
<script src="https://cdn.jsdelivr.net/gh/polskrypt/skrypty/richtexteditor.js"></script>

Usage

Then to create the WYSIWYG editor, paste the following code in the selected place on your website:
<div id="RichTextEditor"></div>
<script>
var textEditor = new kozyrTextEditor(document.getElementById('RichTextEditor'));
</script>
This code will generate the same text editor as in the example above. The editor will appear exactly at the place where you will paste the code. Of course, you can make any changes in the editor and fully adapt it to your needs. A detailed description of using the editor can be found in the documentation below.

Documentation

To create an editor, you need to initialize a new editor object and assign it to any variable:
var textEditor = new kozyrTextEditor(element, settings, css)
As you can see, you can pass three arguments to the editor constructor:
Argument Description
element html DOM element where the text editor is to be created (required).
settings Object containing additional settings for the editor (optional). Possible properties are: 'toolbar', 'items', 'infoBox', 'infos', 'initialHTML' and 'texts'. See below for usage descriptions for all properties.
css String with an additional CSS rules for customize editor's look (optional). You can find a list of the most important CSS classes used in the editor below.

Settings

In the settings object you can customize the text editor with the following properties:
Property Type Description
toolbar String Possible values are top (default), bottom or none. Set where to display the toolbar with text editing tools (none to not display it at all).
items Array An array of text editing tools that will be available in the toolbar. If not defined, all tools will be available. Possible tools include: 'bold', 'italic', 'underline', 'strikethrough', 'group', 'alignLeft', 'alignCenter', 'alignRight', 'justify', 'group', 'orderedList', 'unorderedList', 'indent', 'outdent', 'subscript', 'superscript', 'newLine', 'undo', 'redo', 'clean', 'copy', 'paste', 'cut', 'group', 'background', 'fontColor', 'group', 'removeFormat', 'selectAll', 'horizontalLine', 'save', 'open', 'link', 'image', 'emoticons', 'symbols', 'table', 'print', 'download', 'upload', 'HTMLmode', 'newLine', 'fontName', 'fontSize', 'formatting'
infoBox String Possible values are bottom (default), top or none. Set where to display the box with informations about the text in the editor (none to not display it at all).
infos Array Array of elements to be shown in the infoBox. Possible elements are chars (shows the number of characters in the text), words (shows the number of words in the text) and search (generates a text search field).
initialHTML String Defines an initial HTML string that will be shown in the editor at the start.
texts Object In this object you can translate texts appearing in the editor. As properties, enter the selected texts to be changed, and as their values, the texts that will be displayed instead. The available properties are: 'bold', 'italic', 'underline', 'strikethrough', 'group', 'alignLeft', 'alignCenter', 'alignRight', 'justify', 'group', 'orderedList', 'unorderedList', 'indent', 'outdent', 'subscript', 'superscript', 'newLine', 'undo', 'redo', 'clean', 'copy', 'paste', 'cut', 'group', 'background', 'fontColor', 'group', 'removeFormat', 'selectAll', 'horizontalLine', 'save', 'open', 'link', 'image', 'emoticons', 'symbols', 'table', 'print', 'download', 'upload', 'HTMLmode', 'newLine', 'fontName', 'fontSize', 'formatting', 'chars', 'words', 'search'

Adding own CSS rules

As the third argument, you can pass a string with your own CSS rules to the text editor constructor. Thanks to this, you can give your editor any look you want. The most important classes used by the editor are:
Class Description
.kozyrRichTextEditor Text field in the 'rich text editor' mode.
.kozyrHTMLeditor Textarea in 'HTML' editor mode.
.kozyrToolbar Toolbar with text-editing tools.
.kozyrToolbar .item Class of each text editing item.
.kozyrToolbar .item.active Class of active text editing items.
.kozyrSelect Text editing tools in the form of a select field, such as fontName, fontSize or blockFormat
.kozyrToolsGroup Group of text editing items in the toolbar.
.kozyrSVG SVG icon included in each of toolbar's items.
.kozyrInfoBox Information bar with infos about the text.
.kozyrInfoBox .charsCounterBlock Character counter in the infobox.
.kozyrInfoBox .wordsCounterBlock Word counter in the infobox.
.kozyrInfoBox .searchInput Text field for inline search.
.kozyrInfoBox .searchButton Search button.
If you intend to put more than one editor on one page and use different CSS rules for them, remember to precede each class with the identifier of the element in which the editor is located, so that the given style refers only to the selected editor and not to others.

Methods

The editor object has the following methods:
Method Description
html() Returns the html string currently contained in the editor, or, if you pass it a string as the argument, sets it as the new editor value.
focus() Focuses text editor.
blur() Blur the text editor.
select() Selects the entire contents of a text editor.

Example

Settings

In the example below, we create a text editor assigned to the element with the identifier "RichTextEditor", and pass it the following settings:
<div id="RichTextEditor"></div>
<script>
var textEditorElement = document.getElementById('RichTextEditor'); // we capture the html dom element into which we will place our text editor
var textEditorSettings = { // text editor settings:
toolbar: 'top', // the toolbar is to be placed above the text field
items: ['bold', 'italic', 'underline', 'group', 'alignLeft', 'alignCenter', 'alignRight', 'justify', 'newLine', 'fontName', 'fontSize'], // we list the editing tools to be included in the toolbar
infoBox: 'bottom', // the infoBox is to be placed below the text field
infos: ['chars', 'words', 'search'], // we list the elements to be included in the infoBox
initialHTML: 'Rich Text Editor...', // text that will be the initial value of our editor
texts: { // we translate our editor into Polish:
bold: 'Pogrubienie',
italic: 'Kursywa',
underline: 'Podkreślenie',
alignLeft: 'Wyrównaj do lewej',
alignCenter: 'Wyśrodkuj',
alignRight: 'Wyrównaj do prawej',
justify: 'Wyjustuj',
fontName: 'Czcionka',
fontSize: 'Wielkość czcionki',
chars: 'Znaków',
words: 'Słów',
search: 'Szukaj'
repeats: 'powtórzeń'
}
}
// we create CSS rules to change the appearance of our editor:
var textEditorCSS = `#RichTextEditor .kozyrToolbar, #RichTextEditor .kozyrInfoBox {background:yellow;}
#RichTextEditor .kozyrToolbar .item {background:black;}
`;
var textEditor = new kozyrTextEditor(textEditorElement, textEditorSettings, textEditorCSS); // we create a new editor, passing it as arguments the previously defined values of element, settings and css
</script>
